Installation Steps

Node.js

Your computer must have installed nodejs and yarn to run this template. You can download Node.js from https://nodejs.org and yarn from https://yarnpkg.com/lang/en/docs/install/ . NPM comes bundled with Node.js

Yarn

Install Npm Packages

NPM : You must have to npm install packages. You can do this by running npm install from the root of your project to install all the necessary dependencies.

Running your project

This template runs on Next.js . For detailed instructions on how to use it follow this link https://nextjs.org/docs/


To launch a dev server run npm run dev or yarn dev . Navigate to http://localhost:3000/ . The app will automatically reload if you change any of the source files.


To run a production ready build first run yarn build to build the project then run yarn start to start the server.
